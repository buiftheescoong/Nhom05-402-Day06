AI Learning Co-pilot - 6 Screen Map Prototype

Ban prototype sat tai lieu PRD/SRS: co MVP scope, out-of-scope va flow Happy / Low-confidence / Failure-Correction.

MVP Scope (Hackathon)

IN SCOPE
  • F1: Upload + parse PDF/DOCX + status va recover message.
  • F2: Summary co key points va citation.
  • F3: Quiz tu luan 5-10 cau + nhan xet/cham co ban.
  • F4: Hint 3 cap do (L1/L2/L3), huong tu duy truoc.
  • Feedback loop: Report/Edit + thumbs tai summary/quiz/hint.
  • Grounding: canh bao khi khong du bang chung retrieval.

Out of Scope (Hackathon)

OUT OF SCOPE
  • F5 - Chat Q&A full: de sau MVP.
  • Fine-tuning lon: chua trien khai trong 6 ngay.
  • Advanced OCR: chua support anh mo, cong thuc phuc tap.
  • Curriculum template builder: chua theo day du lop/mon/chapter.

Prototype Flow - Happy Path (sat MVP)

1) Upload User tai PDF/DOCX. doc_id + parse ok
2) Outline Review Kiem tra outline/chunk. Continue -> Summary
3) Summary Doc key points + citation. Click "Tao quiz"
4) Quiz Lam bai va submit. Nhan score + review
5) Hint Bi y -> mo hint L1/L2/L3. Khong dua dap an ngay
6) Feedback Report/Edit neu thay sai. Luu learning signal

Flow sat User Stories (3 lanes)

Happy path Low-confidence fallback Failure + Correction

Lane A - Happy path

S1 Upload ok
S2 Outline dung
S3 Summary ro + citation
S4 Quiz lien quan
S5 Hint huu ich
S6 Hoan thanh + score

Lane B - Low-confidence

S1 Parse 1 phan khong chac
S2 Canh bao vang + cho user sua
S3 Summary gan nhan uncertain
S4 Quiz co tag "tham khao"
S5 Fallback ve doan goc
S6 User tiep tuc hoc

Lane C - Failure + correction

S1 Parse fail/doi dinh dang
S2 User edit outline sai
S3 Bao summary sai
S4 Bao cham quiz sai
S5 Danh gia hint te
S6 Report/Edit -> log
