Prototype Workflow - Core Nghiep Vu

Demo giao dien/chuc nang ung dung theo luong MVP: Upload -> Review -> Summary -> Quiz -> Hint -> Feedback.

Core business flow Happy path Low-confidence fallback Failure + correction

1) Workflow Core (E2E)

S1. Upload Tai Lieu

User tai PDF/DOCX, he thong parse va tao doc_id.

F1

S2. Review Outline

Hien outline/chunk preview, canh bao neu parse chua chac.

Trust Gate

S3. Summary + Citation

Key points + summary body + link Xem doan goc.

F2

S4. Quiz Workspace

Tao 5-10 cau hoi, nhap cau tra loi, cham va nhan xet.

F3

S5. Hint L1/L2/L3

Nhan goi y tang dan, fallback citation khi khong chac.

F4

S6. Feedback/Correction

Report/Edit + thumbs, ghi log de cai thien GAR/TSR.

Learning Loop
S1 -> S2 -> S3 -> S4 -> S5 -> S6

2) Demo Screen Prototype (6 man hinh)

Screen 1 - Upload MVP
Dropzone Keo tha file PDF/DOCX, hien progress.
Validation Kiem tra format, kich thuoc, parse status.
Screen 2 - Outline Review Trust
Outline tree Chapter / section / chunk preview.
Low-confidence panel Highlight doan uncertain + cho sua nhanh.
Screen 3 - Summary MVP
Key Points 3-5 y chinh gon va de nho.
Citation Source chunk ids + nut Xem doan goc.
Screen 4 - Quiz Workspace MVP
Question list 5-10 cau hoi theo scope/chapter.
Answer + Grading Text answer, submit, nhan score + review.
Screen 5 - Hint Drawer MVP
Hint levels L1 goi huong -> L2 thu hep -> L3 gan dap an.
Fallback Neu AI khong chac: tra ve doan goc co lien quan.
Screen 6 - Result + Feedback MVP
Result panel Tong diem, cau sai, recommendation hoc lai.
Correction actions Report/Edit, thumbs up/down, ghi chu loi.

3) Workflow theo tinh huong nghiep vu

Happy path

Upload thanh cong
Outline dung
Summary ro + citation
Quiz dung muc do
Hint huu ich
Feedback tich cuc

Low-confidence fallback

Parse mot phan mo
Canh bao vang + cho sua
Summary gan nhan uncertain
Quiz tag "tham khao"
Hint fallback citation
User tiep tuc hoc an toan

Failure + correction

Upload fail / parse fail
User edit source sai
Report summary sai
Report cham quiz sai
Dislike hint sai huong
Log correction -> improve

4) KPI widget trong demo app

GAR >= 85%

Do tin cay grounding: dung tai lieu + citation hop le.

TSR >= 70%

User giai quyet duoc nhu cau ngay luot dau.

P95 <= 3s

Trai nghiem nhanh cho luong ngan; file lon 5-10 giay.

Out-of-scope hien tai: Chat Q&A full, advanced OCR, curriculum template builder.
